Шаг 2. Подготовьте окружение

Установите 4 инструмента: Python, Docker, Poetry и редактор. Это база, на которой работает всё остальное. Если что-то уже стоит — пропускайте.

1. Python 3.12

Проект работает на Python 3.12. Проверьте текущую версию:

python3 --version

Если версия ниже 3.12 — установите её. Самый чистый способ — через pyenv:

# macOS
brew install pyenv

# Ubuntu/Debian
curl https://pyenv.run | bash

# После установки
pyenv install 3.12.7
pyenv global 3.12.7
Альтернатива

Если не хотите возиться с pyenv — скачайте установщик с python.org.

2. Docker Desktop

Docker нужен для запуска Postgres и Redis в контейнерах. Это база данных и кэш — без них агент не запустится.

  • Скачайте Docker Desktop: docker.com/products/docker-desktop
  • Установите и запустите
  • Проверьте: docker --version и docker compose version
Важно

Docker Desktop должен быть запущен (значок в системном трее). Без него команды docker compose не сработают.

3. Poetry

Poetry — менеджер зависимостей для Python. Установка одной командой:

curl -sSL https://install.python-poetry.org | python3 -

После установки добавьте Poetry в PATH (если установщик не сделал это сам):

# macOS / Linux — добавьте в ~/.zshrc или ~/.bashrc
export PATH="$HOME/.local/bin:$PATH"

Проверка:

poetry --version

4. Редактор кода

Рекомендуем Cursor — это VS Code со встроенным AI. Можно работать с проектом, спрашивать у Claude/GPT прямо в редакторе, генерить код по описанию.

  • cursor.sh — Cursor (рекомендуем)
  • code.visualstudio.com — VS Code (бесплатная альтернатива)

5. Установите зависимости проекта

Из папки AI_sales_write выполните:

poetry install

Poetry прочитает pyproject.toml и установит все библиотеки: LangGraph, FastAPI, SQLAlchemy, mem0, Anthropic SDK и десятки других. Установка занимает 2–5 минут.

Если планируете использовать голосовой канал (Pipecat + Deepgram + ElevenLabs) — добавьте флаг:

poetry install --with voice
Если установка падает на пакете psycopg

Установите системные зависимости:

macOS: brew install libpq

Ubuntu/Debian: sudo apt install libpq-dev build-essential

6. Запустите Postgres и Redis

Это сервисы из docker-compose.yml. Один раз выполните:

docker compose up -d postgres redis

Проверить, что они работают:

docker compose ps

Должны увидеть две строки со статусом Up (healthy).

Чек-пойнт
  • Python 3.12 установлен
  • Docker Desktop запущен
  • Poetry установлен и poetry install прошёл успешно
  • Postgres и Redis работают в Docker (docker compose ps показывает healthy)

Если что-то не так

  • Docker daemon не отвечает — проверьте, что Docker Desktop запущен. На Linux может потребоваться sudo systemctl start docker
  • poetry: command not found — добавьте $HOME/.local/bin в PATH и перезапустите терминал
  • Конфликт версий Python — выполните poetry env use python3.12 в папке проекта
  • Postgres не стартует — посмотрите логи: docker compose logs postgres
