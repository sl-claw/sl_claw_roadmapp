Шаг 3. Получите ключи API

Соберите ключи в одном месте — на следующем шаге вставите их в .env. Обязательны два: OpenAI и Anthropic. Остальные — по нужде.

Обязательные ключи

1. OpenAI API

Используется для embeddings (векторизация текста для RAG-поиска). Бесплатных $5 хватит на ~2 месяца тестов.

  1. Откройте platform.openai.com/api-keys
  2. Зарегистрируйтесь / войдите
  3. Нажмите Create new secret key
  4. Скопируйте ключ (sk-proj-...) — он показывается один раз
  5. В разделе Billing привяжите карту и пополните на $5–10

2. Anthropic API (Claude)

Основная LLM для диалога с клиентом. Claude лучше держит длинный контекст и стиль продавца на русском/украинском.

  1. Откройте console.anthropic.com/settings/keys
  2. Создайте API key
  3. Скопируйте ключ (sk-ant-...)
  4. Пополните баланс ($5 для теста, потом по мере)
Сколько стоит один диалог?

Средний 20-сообщенческий диалог на Claude Sonnet 4.6 = $0.03–0.05. На 1000 диалогов — $30–50. У GPT-4o-mini в 5 раз дешевле, но качество хуже.

Каналы коммуникации

Включаете только нужные. Пустые поля в .env = канал автоматически отключен.

Telegram (рекомендуем для старта)

  1. Откройте @BotFather в Telegram
  2. Введите /newbot
  3. Дайте боту имя и username (должен заканчиваться на bot)
  4. Скопируйте токен (выглядит как 123456:ABC-DEF...)

WhatsApp Cloud API

  1. Откройте developers.facebook.com
  2. Создайте новое приложение → WhatsApp Business → Cloud API
  3. Получите Phone Number ID, Permanent Access Token и App Secret
  4. Verify Token придумайте сами (любая случайная строка)

Instagram Direct

  1. В том же Meta Developer-приложении добавьте Instagram Messaging
  2. Подключите бизнес-аккаунт Instagram (через Facebook Page)
  3. Получите Page Access Token
  4. Verify Token — любая случайная строка

Голос (опционально)

Для голосового канала нужны 3 сервиса:

  • Deepgram — speech-to-text ($200 free credit)
  • ElevenLabs — text-to-speech (10к символов бесплатно)
  • Daily.co — WebRTC транспорт (10к минут бесплатно)

Счёт по реквизитам (без процессинга)

Для B2B-клиентов и тех, кто платит безналом. Anna генерирует PDF-счёт с вашими реквизитами, клиент оплачивает через банк-клиент.

Никаких ключей не нужно — просто подготовьте:

  • Назву ФОП/ТОВ та ЄДРПОУ
  • IBAN рахунку
  • Назву банку та МФО
  • Адресу та контакти

Минимум для старта

Достаточно собрать OpenAI + Anthropic + Telegram BotFather. Этого хватит для первого запуска и первых тестовых диалогов. Платежи и CRM подключите потом.
