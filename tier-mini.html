M1.2 Подготовка машины (Python, Docker, VS Code) 12 мин 🟡 в разработке
Цель: студент устанавливает 4 программы и видит зелёный «healthy» в Docker.
Сценарий: Mac (Homebrew + python@3.12) → Windows (WSL2 + apt) → Docker Desktop → VS Code + 2 расширения → Poetry → проверка `python3 --version && docker --version && poetry --version`.
Студент: параллельно ставит всё. Финальная проверка показывает 3.12.x.
Результат: 4 программы установлены, терминал реагирует на команды.

M1.5 ★ .env + первый запуск (главный wow-урок) 15 мин 🌟 ключевой
Цель: студент видит как Anna отвечает в чате на его команды. Это wow-момент тарифа.
