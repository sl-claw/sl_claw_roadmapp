🆕 Главная новая фича PRO — Niche Marketplace

После оплаты PRO в личном кабинете на token.lux-promo.com/cabinet/niches открывается витрина готовых ниш. Каждая = упакованный набор (persona + каталог + объекции + интеграции). Одной кнопкой импортируется в инстанс студента.

Что внутри одного `.niche` файла

niche-realestate.niche/
├── meta.yaml              ← версия, автор, лицензия, deps
├── persona.py             ← TenantPersona
├── catalog.json           ← структурированный каталог
├── objections.md          ← 30+ возражений
├── spin_questions.md      ← SPIN под нишу
├── meddpicc.md            ← B2B квалификация
├── cases.md               ← реальные кейсы
├── tone.md                ← стилевые особенности
├── cta_phrases.md         ← готовые CTA
├── integrations/          ← нишевые API (например, Bitrix для риэлторов)
├── prompts/               ← кастомные промпт-фрагменты
└── media/                 ← фото товаров, видео-кейсы

3 способа установки в инстанс

  • Через Telegram-бота: команда /install niche-realestate — за 1 минуту установлено и активно
  • Через CLI: sl-claw niche install ~/Downloads/niche-realestate-v1.2.3.niche
  • Через Coolify: переменная INSTALL_NICHES=realestate,construction в env — при деплое подгружается автоматически

Выгрузка собственной настроенной ниши

sl-claw niche export my-tuned-realestate
# → my-tuned-realestate-v1.0.0.niche

Что даёт:

  • Передать .niche своему клиенту (он установит у себя)
  • Сохранить копию для другого клиента в той же нише
  • (Q4 2026) Опубликовать в Marketplace и продавать (70%/30% split)
