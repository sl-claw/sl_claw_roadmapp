Старт за 7 шагов

Идите по порядку. Каждый шаг — отдельная страница, ~5–15 минут на каждый.

  1. 1 Получите ядро репозитория Клонируйте проект с GitHub одной командой
  2. 2 Подготовьте окружение Python 3.12, Docker, Cursor, Poetry
  3. 3 Получите ключи API OpenAI, Anthropic Claude, Telegram, LiqPay/Fondy
  4. 4 Заполните .env Минимальная конфигурация под вашу персону
  5. 5 Первый запуск локально Поднять Postgres + Redis, запустить FastAPI, проверить чат
  6. 6 Настройте свою нишу Persona, каталог, скрипты продаж, база знаний
  7. 7 Запустите в проде Coolify на VPS, домен, webhooks каналов
