Программа курса — 7 шагов

Идите по порядку. Каждый шаг — отдельная страница, ~5–15 минут. Полное прохождение — около часа.

  1. 1 Получите ядро репозитория Клонируйте проект с GitHub одной командой
  2. 2 Подготовьте окружение Python 3.12, Docker, Cursor, Poetry
  3. 3 Получите ключи API OpenAI, Anthropic Claude, Telegram, LiqPay/Fondy
  4. 4 Заполните .env Минимальная конфигурация под вашу персону
  5. 5 Первый запуск локально Postgres + Redis, FastAPI, тест диалога
  6. 6 Настройте свою нишу Persona, каталог, скрипты, база знаний
  7. 7 Запустите в проде VPS, Coolify, домен, webhooks каналов
